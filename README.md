# Promedia Tu semestre

En el actual repositorio encontrará el código y las distintas versiones de una aplicación móvil educativa en la cual usted podrá.

  - Calcular el promedio de un semestre académico
  - Calcular el promedio de la nota de una materia
  - Calcular cuanta nota necesita para aprobar una materia

esto con el añadido de recibir un mensaje e imagen el cual hace ilusión de forma optimista y/o sarcástica de la nota obtenida.
Podrá encontrar la aplicación en la Play Store haciendo clic en el siguiente [enlace](https://play.google.com/store/apps/details?id=mi.aplicacion.PromediaTuSemestre)
